정규표현식

정규표현식(regular expression)은 문자열에서 특정한 규칙을 찾아내어 일치 여부 또는 일치하는 문자열을 알아내거나 치환할 때 사용한다. 정규표현식 객체는 RegExp 생성자를 사용해 만들어 낼 수도 있고 슬래시 사이에 패턴을 두는 /패턴/ 리터럴 방식을 사용해 만들 수도 있다.

자주 사용하는 패턴 문자열

  • [문자]: 각괄호 안에 있는 문자 한 개에 일치한다. 예를 들어 [abc]는 a 또는 b 또는 c 문자를 찾는다.
  • [^문자]: 각괄호 안에 있지 않은 문자 한 개에 일치한다. 예를 들어 [^abc]는 a, b, c가 아닌 문자 한 개에 일치한다.
  • [시작문자-끝문자]: 해당 범위에 있는 문자 한 개에 일치한다. 예를 들어 [0-9]는 숫자 한 개에 일치한다.
  • [^시작문자-끝문자]: 해당 범위에 없는 문자 한 개에 일치한다. 예를 들어 [^0-9]는 숫자가 아닌 문자 한 개에 일치한다.
  • \w: [a-zA-Z0-9_]와 같은 의미이다.
  • \W: [^a-zA-Z0-9_]와 같은 의미이다.
  • \d: [0-9]와 같은 의미이다.
  • \D: [^0-9]와 같은 의미이다.
  • \s: [ \t\n\r]과 같은 의미이다.
  • \S: [^ \t\n\r]과 같은 의미이다.
  • \t: 탭 문자에 일치한다.
  • \r: 캐리지 리턴 문자에 일치한다.
  • \n: 줄바꿈 문자에 일치한다.
  • .: 어떤 문자에든 일치한다.

수량자(quantifier)

  • n*: n 패턴이 0번 이상 반복된 것에 일치한다.
  • n+: n 패턴이 1번 이상 반복된 것에 일치한다.
  • n{A}: n 패턴이 A번 반복된 것에 일치한다.
  • n{A,B}: n 패턴이 A번 이상, B번 이하 반복된 것에 일치한다.
  • n{A,}: n 패턴이 A번 이상 반복된 것에 일치한다.
  • n$: n 패턴이 문자열 끝에 있는 것에 일치한다.
  • ^n: n 패턴이 문자열 시작에 있는 것에 일치한다.

수정자(modifier): 패턴 리터럴을 닫는 슬래시 바로 뒤에 붙인다.

  • g: 전체 탐색 (global)
  • i: 대소문자 구분안함 (ignore cases)
  • m: ^과 $를 줄 시작과 끝에 일치 (multiline)

정규표현식을 사용하는 메소드로는 정규표현식 객체의 test(), exec() 그리고 문자열 객체의 search(), match(), replace(), split()이 있다.

DOM 컬렉션

일부 엘리먼트는 일부러 찾지 않아도 이미 document 객체에 컬렉션 형태로 저장되어 있다.

  • document.anchors: 앵커 엘리먼트의 컬렉션
  • document.forms: 이미지 엘리먼트의 컬렉션
  • document.images: 이미지 엘리먼트의 컬렉션
  • document.scripts: 스크립트 엘리먼트의 컬렉션
  • document.styleSheets: 스타일 엘리먼트의 컬렉션

DOM 이벤트

DOM API를 사용해 이벤트 다루는 법을 살짝 배워봅니다. 이벤트를 추가할 때는 엘리먼트 객체의 addEventListener()와 removeEventListener() 메소드를 사용합니다.

Form 엘리먼트 다루기

Form 요소를 직접 다루어 값을 가져오고 설정하는 법을 배워봅니다.

jQuery: 시작

jQuery는 DOM 작업을 비롯한 여러 자바스크립트 작업을 편리하게 만들어주는 라이브러리이다.
