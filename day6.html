반복문

for

주로 반복 횟수가 정해져 있을 때 사용하는 반복문. 조건식을 만족하는 한 계속 반복한다.

for (초기식; 조건식; 증감식) {
	// 반복할 코드
}

이를 그림으로 나타내면 다음과 같다.

while

주로 반복 횟수가 명확하지 않을 때 사용하는 반복문.

while (조건식) {
	// 반복할 코드
}

이를 그림으로 나타내면 다음과 같다.

내장 타입의 생성자

배열, 문자열, 숫자 등은 각기 해당하는 생성자(constructor)가 있습니다.

내장 객체 확장

prototype를 사용하면 내장 객체의 기능을 확장할 수 있습니다.

사용자정의 타입 작성

지금까지 배운 생성자(constructor)와 prototype를 사용하면 자신만의 타입을 작성할 수 있습니다. 이는 다른 언어의 클래스(class)에 해당하는 기능입니다.
